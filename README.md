## An(other, minimal,) `emacs` cheat sheet

### Accessing menu
```
M-`
```

### Help

|Keystrokes  | Action     |
|------------|------------|
|`C-h k`| Describe command from keystrokes|
|`C-h f`| Describe command from function name|
|`C-h ?`| help suggestions|
|```M-````| access to menu|


### Files

|Keystrokes  | Action|
|------------|------------
|`C-x C-f`   | Create/Open|
|`C-x C-s`   | Save|
|`C-x C-w`   | Save as|
|`C-x C-c`   | Close|
|`C-x i`     | Insert|

### Moving

|Keystrokes  | Action|
|------------|------------
|`C-p `      | previous line|
|`C-n `      | next line|
|`C-a `      | line beginning|
|`C-e `      | end of line|
|`C-f `      | forward a character|
|`C-b `      | backward a character|
|`M-f `      | forward a word|
|`M-b `      | backward a word|
|`M-> `      | end of buffer|
|`M-< `      | beginning of buffer|

### Select region

|Keystrokes  | Action|
|------------|------------
|`C-space`     | by line
|`C-x space`   | by column

### Editing

|Keystrokes  | Action|
|------------|------------
|`C-w `      | Cut region|
|`M-w`       | Copy region|
|`C-y`       | Paste|
|`C-x u`     | Undo|
