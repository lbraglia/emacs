##An(other, minimal,) `emacs` cheat sheet

### Display menu
`F10`

### Files

|Keystrokes  | Action|
|------------|------------
|`C-x C-f`   | Create/Open|
|`C-x C-s`   | Save|
|`C-x C-c`   | Close|

### Editing

|Keystrokes  | Action|
|------------|------------
|`C-w `      | Cut|
|`M-w`       | Copy|
|`C-y`       | Paste|
|`C-x u`     | Undo|

### Select region

|Keystrokes  | Action|
|------------|------------
|C-space     | by line
|C-x space   | by column
